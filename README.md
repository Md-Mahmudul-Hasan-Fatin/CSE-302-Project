# CSE-302-Project
Project Name - 'Smart Classroom Allocation System'
A web-based classroom management system built as a Database Management course project.

## Technologies Used
- Backend: PHP
- Frontend: HTML, CSS (for GUI and styling)
- Database: MySQL
- Server: XAMPP (MySQL)
- Tool: phpMyAdmin, VS Code

## Features
- Add & Edit & Delete Classroom
- Add & Edit & Delete Course
- Allocate Classroom
- View Allocation
- Login / Logout System
- Role : User & Admin

## How to Run

1. Download and install *XAMPP*
2. Open XAMPP Control Panel → Start *MySQL*
3. Go to *phpMyAdmin* (http://localhost/phpmyadmin)
4. Create the database and tables manually as per the project
5. Clone/download this repository
6. Copy all project files into:
   *C: > xampp > htdocs > smart_classroom*
7. Open any browser and go to:
   *http://localhost/smart_classroom/index.php*
